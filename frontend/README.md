# Frontend (Angular)

Ez a mappa a Todo alkalmazás Angular frontendet tartalmazza.

## Futtatás fejlesztői módban

Előfeltétel: Node.js + npm.

```bash
npm.cmd install
npm.cmd start
```

- UI: http://localhost:4200

A teljes stack futtatásához a repo gyökerében lévő `docker compose up --build` az ajánlott (frontend + nginx + backend + MongoDB).

## Build

```bash
npm.cmd run build
```
